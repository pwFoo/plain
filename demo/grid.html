Grid Demo (h2.full - grid / flex item width 100% itself!)

Easiest way is to work with percent width to be sure to fill the row as expected, because flexible items will shrink as long as the content fit the current row.
Gutters / margin with auto flex items (no width defined) works fine if the following item is full row width to ensure the line break.
Items with defined width have margin between the items by default to fill the row.
"Forced line breaks" need additional markup (nested grid OR a empty .break element like <div> class="break"></div>) :(
